# 🛒 E-Commerce Website

A full-stack e-commerce web application developed as a university project.  
The system allows users to browse products, manage shopping cart, and place orders, while admins can manage products, users, and orders.

---

## 🚀 Features

- User registration and login system  
- Product browsing and search  
- Product details page  
- Shopping cart system  
- Checkout and order processing  
- Order history for users  
- Admin dashboard  
- Product management (Add / Edit / Delete)  
- User management (Admin)  
- Contact form  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  
- XAMPP (Local Development Server)

## ⚙️ How to Run the Project

1. Install XAMPP  
2. Move project folder to:
  C:\xampp\htdocs\
3. Start Apache & MySQL from XAMPP Control Panel  
4. Import database:
- Open phpMyAdmin  
- Create database: `ecommerce_db`  
- Import file: `ecommerce_db.sql`  
5. Open browser:
  http://localhost/Alaa_Hijazi/Project_Ecommerce
